BLIT386 is developed in the [blit386 engine repository](https://github.com/blit386/blit386) on GitHub. Contributions -
issues, pull requests, and documentation fixes - are welcome there.

This site publishes the **public** API reference and guides, which are generated from the engine repository's `docs/`.
Contributor-facing documentation is not mirrored here; it lives alongside the code so it stays versioned with the
engine.

## Improving these docs

The pages on this site are a generated mirror of the engine docs. To fix a public API or guide page, edit its source in
`blit386/docs/` and open a pull request there - the change is regenerated onto this site. See the
[engine documentation](https://github.com/blit386/blit386/tree/main/docs) for the full set, including contributor-only
pages.

## Reporting security issues

Report vulnerabilities privately through GitHub Security Advisories rather than public issues. See the engine repo's
security policy and the
[security runbook](https://github.com/blit386/blit386/blob/main/docs/security/security-runbook.md).
