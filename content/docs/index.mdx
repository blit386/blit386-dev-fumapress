This is the public documentation for the BLIT386 engine. Start with [API: Core](/docs/api/core) for `bootstrap()`, the
game loop, camera, `Timer`, and the core types. The pages below mirror the canonical engine documentation maintained in
the [blit386 repository](https://github.com/blit386/blit386).

Migration is in progress: pages already on this site link here; pages not yet mirrored link to their source on GitHub
and will move here as they are ported.
