This is the public documentation for the BLIT386 engine. New here? Start with [Getting Started](/docs/getting-started).
For the API, see [API: Core](/docs/api/core) for `bootstrap()`, initialization, and the configure surface. The pages
below mirror the canonical engine documentation maintained in the
[blit386 repository](https://github.com/blit386/blit386).
