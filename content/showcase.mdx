Projects built with the BLIT386 engine. If you have built something with BLIT386 and want it listed here, see
[How to submit](#how-to-submit) below.

## Featured projects

### Catcher

The starter game included in every scaffolded project. Demonstrates palette-first rendering, integer coordinates, sprite
animation, and keyboard input. Source code is generated by [create-blit386](https://github.com/blit386/create-blit386)
and ships inside every new project.

### Interactive demos

The official demo suite at [demos.blit386.dev](https://demos.blit386.dev) covers every engine subsystem: palette
presets, post-process CRT effects, bitmap fonts, pointer and gamepad input, the overlay HUD, and more. Each demo is a
single self-contained JavaScript file in the [blit386-demos](https://github.com/blit386/blit386-demos) repository.

## How to submit

To add your project to this page:

1. Open an issue on [github.com/blit386/blit386](https://github.com/blit386/blit386/issues) with the title "Showcase:
   [your project name]".
2. Include a short description (one or two sentences), a link to the project or its source code, and a screenshot or a
   short video clip if available.
3. Projects must use a published version of BLIT386 (`>= 1.0.0`) and must not contain content that violates the
   [code of conduct](https://github.com/blit386/blit386/blob/main/CODE_OF_CONDUCT.md).

There is no formal review SLA for small projects, but a maintainer will respond within a few days.
